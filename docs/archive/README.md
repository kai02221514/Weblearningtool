# アーカイブ

このディレクトリは、完了済み監査、過去時点のレビュー、判断前の証跡を保存する場所である。現時点では主に完了済み監査を保存しているが、過去レビューや監査時点の証跡を置く場合がある。

- アーカイブ文書は履歴資料であり、現在の研究仕様、実装状態、作業優先順位の正本ではない。
- 現在の研究仕様は `../research/`、現在の技術仕様は `../architecture/`、現在の運用手順は `../operations/` を参照する。
- 現在の作業管理はLinearを参照する。
- アーカイブ文書は原則として更新しない。当時のパス表記や前提が残る場合がある。
- アーカイブ内容を現行仕様へ採用する場合は、Decision Logを経由して正本文書へ反映する。

## 更新可能期間

- 監査・レビューの実施中は、対象ディレクトリ内の成果物を更新してよい。
- 監査完了または完了Decision記録後は、原則として内容を凍結する。
- 完了後に訂正が必要な場合は、原文を無断で書き換えず、訂正記録または新しい監査文書として残す。
